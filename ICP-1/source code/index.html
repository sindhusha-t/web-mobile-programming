GitHub Documentation

GitHub Introduction:

GitHub is a distributed Version control system (also known as Source code management) used for tracking changes to the file(s).

  • GitHub provides a platform for multiple developers to work simultaneously.
  • GitHub adds flexibility to developers to keep track of file changes locally during offline and can later push changes to server when required.
  • GitHub is heavily used for open source projects.

GitHub Installation:

  1. Signing up in the GitHub website

  2. Logging into the github website using the Signed up account.

  3. Downloaded Git Bash and Installed - it is a command line console for accessing the Github remote respository.

  4. Downloaded and Installed Git Desktop and logged into the account
  5. One can create a new repository in the github website or from either git bash or git desktop.

  6. Cloning the remote repository in the local repository

  7. Commands for pushing the files to the remote repository
